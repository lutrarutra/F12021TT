F1 2021 Time Trial Analysis

u/AsfahanM

What?

Welcome to my data mining project. On this website I present my findings from analysing 50k lap times on every circuit in the F1 2021 Game. Lap times are from players on PC platform in dry weather conditions. The motivation of this project is my curiosity as well as a hope to gain insights from the data gathered. This project can serve as point of reference when for example discussing the impact of assists or comparing your times to the quickest drivers in the world.

How?

As you may know, the leaderboards are only available through in game scrollable menu. This provides a very limited access to the data. In order to adress this issue, I implemented a rather simple data mining pipeline. In short, the pipeline constists of following steps:

  • 1. Screen record scrolling through 50k lap times on every circuit of F1 2021 game.
  • 2. Document the lap times by processing the recording frame by frame with Tessaract, an optical character recognition (OCR) tool.
  • 3. Filter, analyse, and plot the documented lap times.

It is important to note that the data mining pipeline is not perfect by any means;in some cases Tessaract was not successful in recognising the text which results in missing some of the lap times.

Source code is available in the Github repository of this project.
